# Listing Rewrite Prompt

Rewrite titles into safer, neutral, non-misleading wording for preliminary IP risk screening.

Rules:
- Remove style / inspired by / look alike.
- Remove fake / replica / dupe.
- Replace "compatible with" with more cautious wording.
- If has_authorization=false, avoid putting brand terms in title core.
- Avoid official / authorized / genuine / authentic.
- Never help evade legal liability or disguise infringement.
